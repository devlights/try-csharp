# IDE
- JetBrains Rider
- Microsoft VisualStudio

# Run (IDE)
+ set TryCSharp.Tools.Cui project as 'startup project'
+ run onto IDE

# Run (CLI)

```sh
$ cd TryCSharp.Tools.Cui
$ dotnet run
```

or 

```sh
$ cd TryCSharp.Tools.Cui
$ dotnet run --onetime
```
